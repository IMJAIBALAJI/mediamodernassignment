    • What is HTML5

    HTML5 is the latest version of the HyperText Markup Language, the code that describes web pages. It introduces new elements and attributes that make it easier to create web pages with better structure and presentation.

    What is Semantic HTML

    Semantic HTML refers to the use of HTML tags that convey the meaning (semantics) of the content within them. This improves the readability of the code and helps search engines and assistive technologies understand the content better.

    Common Semantic Tags

    • <header> - Defines the header of a section or page.
    • <nav> - Defines a block of navigation links.
    • <main> - Defines the main content of the document.
    • <article> - Defines an independent, self-contained piece of content.
    • <section> - Defines a section within a document.
    • <footer> - Defines the footer of a section or page.
