Differences between HTML and HTML5

Feature Difference
Semantic Elements HTML uses generic tags like <div> and <span> while HTML5 introduces semantic tags like <header>, <footer>, and <article>.
Multimedia Support HTML requires plugins for multimedia, while HTML5 has native support for <audio> and <video> elements.
APIs and Features HTML lacks built-in advanced APIs, while HTML5 includes APIs like Canvas (<canvas>), Geolocation, and Web Storage (localStorage and sessionStorage).
